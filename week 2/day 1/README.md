# What is LWC?

Lightning Web Components (LWC) is Salesforce’s modern framework for building fast, reusable, and scalable user interfaces using:
- HTML
- JavaScript
- CSS

LWC is based on modern web standards and provides better performance than older frameworks.

---

# Why Salesforce Uses LWC

Salesforce moved toward LWC because:
- Faster rendering
- Better performance
- Modern JavaScript support
- Easier maintenance
- Better scalability
- Improved security
- Reusable component architecture

---

# Lightning Web Components Basics

Each LWC component contains:
- HTML file
- JavaScript file
- Meta XML configuration file

These files together define:
- UI structure
- Component logic
- Salesforce exposure settings

---

# Component-Based UI Architecture

A component is a reusable piece of UI.

Examples:
- Header
- Navigation menu
- Notification panel
- Attendance card

Benefits:
- Reusability
- Faster development
- Easier maintenance
- Better scalability
- Improved team collaboration

# Frontend vs Backend Logic

| Functionality | Frontend | Backend |
|---|---|---|
| Button Click | Yes | No |
| Notification Display | Yes | No |
| UI Rendering | Yes | No |
| Data Validation | Yes | Yes |
| Fee Calculation | No | Yes |
| Database Save | No | Yes |
| Security Checks | No | Yes |
