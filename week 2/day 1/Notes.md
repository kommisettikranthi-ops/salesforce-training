# 1. Introduction to Lightning Web Components (LWC)

## What is LWC?

Lightning Web Components (LWC) is Salesforce’s modern JavaScript framework used to build fast, reusable, and scalable user interfaces on the Salesforce platform.

LWC is built using:
- HTML
- JavaScript
- CSS
- Modern Web Standards

Unlike older technologies, LWC uses native browser features which improves performance and development experience.

---

## Why LWC is Important

LWC is important because enterprise applications require:
- Fast UI rendering
- Reusable components
- Better scalability
- Easier maintenance
- Clean architecture
- Modular development

LWC helps developers create applications that are:
- Efficient
- Reusable
- Secure
- Easy to maintain
